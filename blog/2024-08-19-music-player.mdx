# Music Player 播放器

用 React 結合 Web Audio Api 製作的一個音樂播放器

Features:

  1. 播放 (從暫停時間開始)
  2. 暫停
  3. 停止
  4. 調整音量
  5. 排序歌曲
  6. 隨機排序歌曲
  7. 顯示正在播放歌曲資訊
  8. 刪除歌曲
